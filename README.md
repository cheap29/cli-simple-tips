# CLI Tips - コマンドラインチートシート

> Windows・Linux・Git・PowerShell のコマンドをキーワード検索・フィルタリングしてすぐ参照できる Web アプリです。

ブラウザで `index.html` を開くだけで動作します。サーバー不要・フレームワーク不要。

---

## 機能

- **リアルタイム検索** — コマンド名・説明の部分一致で即座に絞り込み
- **OS/ツール別フィルタ** — Windows / Linux / Git / PowerShell を切り替え
- **詳細展開** — 使用例・パラメータ説明を1件ずつ表示
- **ワンクリックコピー** — コマンドをクリップボードに即コピー
- **全コマンド一覧** — フローティングボタンからカテゴリ別一覧を表示
- **オフライン動作** — ローカルファイルで完結

---

## 使い方

1. `index.html` をブラウザで開く
2. 検索ボックスにキーワードを入力、またはフィルタボタンで OS を選択
3. 「詳しく」ボタンでパラメータ詳細を確認
4. コマンドをクリックまたはコピーボタンで取得

---

## コマンドの追加

`data/commands.json` にエントリを追加してブラウザをリロードするだけです。

```json
{
  "command": "dir",
  "example": "dir /a",
  "description": "ディレクトリの内容を表示",
  "parameters": [
    { "param": "/a", "description": "隠しファイルも含めて表示" }
  ]
}
```

## 技術仕様

- 純粋な HTML / CSS / JavaScript（ビルド不要）
- `navigator.clipboard` API（フォールバック: `execCommand`）
- レスポンシブ対応（1〜3カラム）
