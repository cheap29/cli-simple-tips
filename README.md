# 🖥️ CLI Tips - コマンドラインチートシート

Windows・Linux・Git・PowerShell のコマンドを効率的に検索・参照できる Web アプリケーション

## 📖 概要

Windows、Linux、Git、PowerShell のコマンドを簡単に検索・参照できる Web アプリケーションです。リアルタイムフィルタリング、詳細パラメータ表示、ワンクリックコピー機能を搭載しています。

## ✨ 主な機能

### 🔍 **リアルタイム検索・フィルタリング**

- テキスト入力による即座の検索
- OS・ツール別フィルタリング（Windows、Linux、Git、PowerShell）
- 複数条件の組み合わせ検索

### 📋 **コマンド詳細表示**

- コマンドの使用例とパラメータ説明
- 排他的詳細展開（1 つずつ表示）
- コードスニペットのワンクリックコピー

### 🎨 **モダン UI**

- Material Design インスパイアのデザイン
- 3 カラムレスポンシブレイアウト
- OS・ツール別カラーコーディング
- アクセシビリティ対応

### 📱 **全コマンド一覧**

- フローティングボタンからアクセス
- カテゴリ別グループ表示
- クリックでコピー＆詳細表示にジャンプ

## 🚀 使い方

### 1. **基本的な使い方**

1. ブラウザで `index.html` を開く
2. 検索ボックスにキーワードを入力
3. フィルタボタンで OS・ツールを選択
4. 「詳しく」ボタンでパラメータ詳細を表示

### 2. **コピー機能**

- **詳細表示内**: 「コピー」ボタンでコマンドをクリップボードに
- **全コマンド一覧**: コマンドをクリックでコピー＆詳細表示

### 3. **キーボードナビゲーション**

- `Tab` / `Shift+Tab`: フォーカス移動
- `Enter` / `Space`: ボタン実行
- `Escape`: 詳細表示・モーダルを閉じる

## 📊 データ構造

### **commands.json**

```json
{
  "commands": [
    {
      "command": "dir",
      "description": "ディレクトリの内容を表示",
      "example": "dir /a",
      "parameters": [
        {
          "param": "/a",
          "description": "隠しファイルも含めて表示"
        }
      ]
    }
  ]
}
```

## 🎯 主要機能の実装

### **リアルタイムフィルタリング**

- テキスト検索: コマンド名・説明での部分一致
- OS 判定: コマンド名のプレフィックス自動判定
- 複数条件: AND 条件での絞り込み

### **排他的詳細展開**

- 1 つの詳細を開くと他は自動で閉じる
- スムーズなアニメーション
- 状態管理による一貫性

### **クリップボード API**

- モダンブラウザ: `navigator.clipboard.writeText()`
- フォールバック: `document.execCommand('copy')`
- 視覚的フィードバック

## 🔧 開発・カスタマイズ

### **新しいコマンド追加**

1. `data/commands.json` にコマンド情報を追加
2. 必要に応じて OS 判定ロジックを調整
3. ブラウザをリロードして確認

### **スタイルカスタマイズ**

- `css/styles.css` の CSS 変数を編集
- カラーパレット・フォント・サイズを調整
- レスポンシブブレークポイントの変更

### **機能追加**

- `js/app.js` に新機能を追加
- モジュール化されたコードで拡張しやすい設計
- イベントハンドラーベースのアーキテクチャ
